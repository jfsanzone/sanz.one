# Dark Mono Design System — sanz.one

## Color Tokens
- `--bg`: #080808 (near-black)
- `--bg-surface`: #111111 (slightly lifted surface)
- `--text`: #f0f0f0 (near-white primary)
- `--text-body`: rgba(240,240,240,0.75) (body copy)
- `--text-muted`: rgba(240,240,240,0.40) (metadata, labels)
- `--border`: rgba(240,240,240,0.07) (hairline dividers)
- `--accent`: #f0f0f0 (monochrome hover — white)
- `--accent-dim`: rgba(240,240,240,0.12) (subtle highlight bg)

## Typography
- Display/UI: Space Mono 700 — uppercase, tracked
- Body: DM Sans 400/500 — readable prose
- Mono labels: Space Mono 400 — dates, breadcrumbs, meta

## Design Principles
- Locked dark — no light mode toggle (pure dark mono)
- Grain texture overlay via CSS SVG filter for depth
- Hairline borders only (1px, very low opacity)
- Generous whitespace, narrow max-width columns
- Hover states: white color reveal, no color accents
- Subtle fade-in entrance animations
- Social icons: monochrome, white on hover only
