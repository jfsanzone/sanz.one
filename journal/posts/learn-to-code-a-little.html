Learn to Code a Little

January 3, 2026

I'm not a developer. I want to be clear about that. But I can read HTML, understand how CSS works, and fumble my way through enough JavaScript to build simple things. And that "enough" has been more valuable to my design career than any Figma plugin.

When you understand how things get built, you design differently. You stop proposing interactions that sound cool in a presentation but are nightmares to implement. You start thinking about states, edge cases, and data. Your designs get more grounded because they're informed by how the medium actually works.

With AI, the bar dropped even lower. I can describe what I want in plain language and get working code back. I still need to understand enough to know when the code is wrong or when the approach doesn't make sense. But the gap between "designer who can't code" and "designer who ships real things" has never been smaller. There's no excuse not to close it.
